# Car Price Prediction #

## Project Overview
This project predicts the selling price of a used car using Machine Learning.

## Technologies Used
Python
Pandas
Scikit-Learn
Random Forest Regressor

## Features
Year
Present_Price
Driven_kms
Fuel_Type
Transmission
Owner

## Model Performance
